Detailed Description

This internal handler implements downgrade from SL_ERROR to SL_WARNING for statements which support IGNORE.

This handler is used for the statements which support IGNORE keyword. If IGNORE is specified in the statement, this error handler converts the given errors codes to warnings. These errors occur for each record. With IGNORE, statements are not aborted and next row is processed.
