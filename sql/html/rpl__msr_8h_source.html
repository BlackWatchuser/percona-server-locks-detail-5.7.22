404 static bool inline is_slave_configured()
405 {
406  /* Server was started with server_id == 0
407  OR
408  failure to load slave info repositories because of repository
409  mismatch i.e Assume slave had a multisource replication with several
410  channels setup with TABLE repository. Then if the slave is restarted
411  with FILE repository, we fail to load any of the slave repositories,
412  including the default channel one.
413  Hence, channel_map.get_default_channel_mi() will return NULL.
414  */
