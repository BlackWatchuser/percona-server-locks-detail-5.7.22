Detailed Description

Represents a MySQL value opaquely, i.e. the Json DOM can not serialize or deserialize these values. This should be used to store values that don't map to the other Json_scalar classes. Using the "to_string" method on such values (via Json_wrapper) will yield a base 64 encoded string tagged with the MySQL type with this syntax:

"base64:typeXX:<base 64 encoded value>"
