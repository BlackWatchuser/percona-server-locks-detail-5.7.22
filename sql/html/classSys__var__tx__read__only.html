Detailed Description

Class representing the tx_read_only system variable for setting default transaction access mode. This variable is deprecated and will be removed in future release. 'transaction_read_only' is used as an alternative instead.

Note that there is a special syntax - SET TRANSACTION READ ONLY (or READ WRITE) that sets the access mode for the next transaction only.
