64 /*
65  Temporary table used by semi-join DuplicateElimination strategy
66 
67  This consists of the temptable itself and data needed to put records
68  into it. The table's DDL is as follows:
69 
70  CREATE TABLE tmptable (col VARCHAR(n) BINARY, PRIMARY KEY(col));
71 
72  where the primary key can be replaced with unique constraint if n exceeds
73  the limit (as it is always done for query execution-time temptables).
74 
75  The record value is a concatenation of rowids of tables from the join we're
76  executing. If a join table is on the inner side of the outer join, we
77  assume that its rowid can be NULL and provide means to store this rowid in
78  the tuple.
79 */
