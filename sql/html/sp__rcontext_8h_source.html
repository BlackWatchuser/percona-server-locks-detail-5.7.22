34 /*
35  This class is a runtime context of a Stored Routine. It is used in an
36  execution and is intended to contain all dynamic objects (i.e. objects, which
37  can be changed during execution), such as:
38  - stored routine variables;
39  - cursors;
40  - handlers;
41 
42  Runtime context is used with sp_head class. sp_head class is intended to
43  contain all static things, related to the stored routines (code, for example).
44  sp_head instance creates runtime context for the execution of a stored
45  routine.
46 
47  There is a parsing context (an instance of sp_pcontext class), which is used
48  on parsing stage. However, now it contains some necessary for an execution
49  things, such as definition of used stored routine variables. That's why
50  runtime context needs a reference to the parsing context.
51 */
