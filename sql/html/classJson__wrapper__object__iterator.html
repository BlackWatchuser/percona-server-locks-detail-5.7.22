◆ elt()

std::pair< const std::string, Json_wrapper > Json_wrapper_object_iterator::elt ( ) const

Get the current element as a pair: the first part is the element key, the second part is the element's value. If the underlying object is a DOM, the returned wrapper for the element will hold an alias for the element DOM, that is, no copy of the DOM will be taken. In other words, the ownership of the element DOM still resides with whoever owns the DOM of which the element is part.
