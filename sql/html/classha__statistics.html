double ha_statistics::table_in_mem_estimate

Estimate for how much of the table that is availabe in a memory buffer. Valid range is [0..1]. If it has the special value IN_MEMORY_ESTIMATE_UNKNOWN (defined in structs.h), it means that the storage engine has not supplied any value for it.
