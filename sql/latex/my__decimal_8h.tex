\subsection{Detailed Description}
It is interface module to fixed precision decimals library.

Most functions use \textquotesingle{}uint mask\textquotesingle{} as parameter, if during operation error which fit in this mask is detected then it will be processed automatically here. (errors are E\+\_\+\+D\+E\+C\+\_\+$\ast$ constants, see include/decimal.\+h)

Most function are just inline wrappers around library calls
