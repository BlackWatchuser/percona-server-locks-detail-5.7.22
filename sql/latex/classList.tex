Sort the list. 


\begin{DoxyParams}{Parameters}
{\em cmp} & node comparison function \\
\hline
{\em arg} & additional info to be passed to comparison function\\
\hline
\end{DoxyParams}
The function sorts list nodes by an exchange sort algorithm. The order of list nodes isn\textquotesingle{}t changed, values of info fields are swapped instead. Due to this, list iterators that are initialized before sort could be safely used after sort, i.\+e they wouldn\textquotesingle{}t cause a crash. As this isn\textquotesingle{}t an effective algorithm the list to be sorted is supposed to be short.
