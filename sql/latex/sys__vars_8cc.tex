\subsection{Detailed Description}
Definitions of all server\textquotesingle{}s session or global variables.

How to add new variables\+:


\begin{DoxyEnumerate}
\item copy one of the existing variables, and edit the declaration.
\item if you need special behavior on assignment or additional checks use O\+N\+\_\+\+C\+H\+E\+CK and O\+N\+\_\+\+U\+P\+D\+A\+TE callbacks.
\item {\itshape Don\textquotesingle{}t} add new Sys\+\_\+var classes or uncle Occam will come with his razor to haunt you at nights
\end{DoxyEnumerate}

Note -\/ all storage engine variables (for example myisam\+\_\+whatever) should go into the corresponding storage engine sources (for example in storage/myisam/ha\+\_\+myisam.\+cc) !
